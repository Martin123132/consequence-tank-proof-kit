# Access Request

Use this page when requesting access to the private Consequence Tank engine, private release artifacts, implementation internals, benchmark internals, or a commercial evaluation.

Private access is not automatic. Access may require identity verification, intended-use review, written permission, and an NDA.

## Request Checklist

Include:

- your name and organisation
- contact email
- GitHub username
- intended use
- whether the use is personal, research, nonprofit, commercial evaluation, or production commercial
- whether you need source access, private build access, benchmark review, technical walkthrough, or hosted demo
- whether your organisation will sign an NDA before access
- whether any model provider, search company, AI platform, analytics product, logistics product, hosted service, or commercial product will see or use the work

## What May Be Shared

If approved, access may be limited to one or more of:

- private repository review access
- private release artifact review
- private benchmark evidence review
- private technical walkthrough
- pilot evaluation materials
- commercial diligence materials

## What Is Not Granted By Access

Private access does not automatically grant:

- commercial use
- production use
- sublicensing
- redistribution
- hosted-service rights
- model-training rights
- patent rights
- trademark rights
- rights to publish private files, screenshots, benchmark internals, or implementation details

Those rights require a separate written agreement.
