# Access Request Checklist

Before requesting private access, prepare:

- name
- organisation
- contact email
- GitHub username
- intended use
- access type requested
- commercial or noncommercial status
- whether an NDA can be signed
- whether a model provider, AI platform, hosted service, search company, analytics product, logistics product, or commercial product will see or use the work

Private access may be refused, limited, time-bounded, or revoked.

Commercial use requires separate written permission.
