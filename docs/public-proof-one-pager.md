# Consequence Tank Public Proof One-Pager

## Product Shape

Consequence Tank is a private-core browser app for consequence-aware agent evaluation. It makes a warehouse-style agent run inspectable, then carries the result through Analytics, Evidence, and Audit review.

## Public Claim

The private `v0.1-private-alpha` checkpoint demonstrates a guided local workflow:

1. Run or restore a consequence simulation.
2. Compare saved experiment evidence.
3. Save report-pack decisions and handoffs.
4. Verify decision packages, governance ledger state, and audit bundles.

## Public Proof

- Private-alpha validation passed.
- The guided Evidence Path is visible on desktop and mobile.
- The flow has Run, Compare, Decide, and Audit phases.
- Browser QA showed clean console warning/error logs.
- Mobile QA showed no page-level horizontal overflow.

## Boundary

This proof kit does not publish source code, implementation internals, private benchmark configuration, private release artifacts, or commercial rights.

Private access requires written permission and may require an NDA.
