# Public Positioning

Consequence Tank should be described publicly as a private-core proof-backed product, not as an open-source code release.

## Short Description

Consequence Tank is a private-core browser technology for making consequence-aware agent behaviour inspectable from simulation through evidence and audit.

## What Makes It Different

It is not only a simulation.

It is not only an analytics dashboard.

It is not only an audit log.

The private engine connects these into one guided review path:

```text
Simulation -> Analytics -> Evidence -> Audit
```

## Public Boundary

The public proof kit shows screenshots, proof cards, validation summaries, and the access path.

It does not reveal source code, implementation details, private benchmark internals, generated evidence, or commercial rights.

## Reviewer Call To Action

Review the proof kit. If the work is relevant, request private access through the access path and be ready to sign an NDA if requested.
