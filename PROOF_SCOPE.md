# Proof Scope

This proof kit is a public evidence wrapper around a private engine.

It is designed to prove that the project exists, has a working private-alpha evidence story, and has inspectable guided UX worth evaluating. It is not designed to reveal implementation internals.

## Public In This Repository

- product description
- evidence-card summaries
- validation result headlines
- screenshots of the guided Analytics path
- access request workflow
- commercial boundary
- public-materials licence

## Not Public In This Repository

- engine source code
- app implementation internals
- package internals
- private release artifacts
- benchmark implementation
- governance ledger implementation
- verifier implementation details
- unpublished notebooks
- private GitHub workflow details
- private customer or pilot material

## Why This Boundary Exists

The public proof kit gives reviewers enough to evaluate the shape of the technology without turning the engine into a public code drop.

Commercial use, private repo access, and implementation review must happen through written permission.
