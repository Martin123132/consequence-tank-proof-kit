# Warehouse Tank Evidence Card

## What It Shows

The private engine demonstrates a consequence-aware warehouse simulation and evidence workflow.

The product flow is:

```text
Simulation -> Analytics -> Evidence -> Audit
```

## Headline Result

Private alpha.

## Public-Safe Details

- Simulation is used as the starting workspace.
- Analytics compares consequence-aware modes and saved experiment evidence.
- Evidence exports support report-pack review and decisions.
- Audit surfaces include evidence chain review, decision package verification, governance ledger review, and bundle verification.

## Boundary

This public card does not publish simulator code, route-control implementation, benchmark internals, private report-pack data, governance internals, or generated private evidence files.
